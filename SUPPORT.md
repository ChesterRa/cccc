# Support

## Usage Questions

For usage and configuration questions:

- Read docs: `docs/`
- Check CLI help: `cccc --help`
- Check environment: `cccc doctor`

## Bug Reports

Use GitHub Issues and include:

- Version (`cccc version`)
- OS/runtime details
- Exact command and output
- Reproduction steps

## Feature Requests

Open a GitHub Issue with:

- Problem statement
- Proposed behavior
- Expected ROI and operational impact

## Operational Notes

This project is local-first. Runtime state is stored under `CCCC_HOME` (default `~/.cccc/`), not in repository files.
